# 🌀 홀로그래픽 디스플레이 구현 (CGH + SLM)

Computer Generated Hologram 기반 홀로그래픽 디스플레이 실현을 목표로 한 **개인 학습 포트폴리오**

## 📚 학습 경로 (5단계)

### Stage 1️⃣: Python 기초
기본 출력, 반복문, 자료형 학습  
NumPy, Pandas, Matplotlib 활용

[자세히 보기](./portfolio/01_Python-Basics/)

### Stage 2️⃣: 회절 시뮬레이션
Fraunhofer 회절 원리 학습  
간섭무늬 계산 및 시각화

[자세히 보기](./portfolio/02_Diffraction-Simulation/)

### Stage 3️⃣: 1D CGH
두 점광원의 간섭 시뮬레이션  
1D Phase-only CGH 구현

[자세히 보기](./portfolio/03_1D-CGH/)

### Stage 4️⃣: 2D CGH
단일 점광원 → 다중 점광원  
2D Phase-only CGH 구현

[자세히 보기](./portfolio/04_2D-CGH/)

### Stage 5️⃣: 위상 정규화 & 저장
위상 정규화 (0~2π → 0~255)

[자세히 보기](./portfolio/05_Phase-Normalization-and-Storage/)
